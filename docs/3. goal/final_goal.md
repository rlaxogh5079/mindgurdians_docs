# 최종 목표

## 연구 최종 목표

```
치료 접근에 어려움이 있는 우울증 환자를 위한 스마트 밴드 기반 원격진료 멀티 플랫폼 애플리케이션 개발
```

| 년도     | 내용                                                                                                                                                                            |
| -------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 1차 년도 | 스마트밴드 기반 생체신호 수집 및 모니터링 시스템 개발 <br /> - 웨어러블 기기를 통해 주요 생체신호를 안정적으로 수집 <br /> - 실시간 모니터링 및 데이터 처리 파이프라인 구현     |
