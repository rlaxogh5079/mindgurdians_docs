## 설정 근거

| 평가항목                            | 목표 설정 근거                                                                                                                        |
| ----------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------- |
| (R&D) 화상통화 지연율               | 실시간 상호작용의 품질을 보장함. 응답 타이밍이 어긋나면 병력 청취·관찰 신뢰도가 떨어지며, 이는 진단 오판 위험을 증가시킴.             |
| (R&D)모니터링 시스템 지연율         | 생체신호 수집 → 저장 → 분석 → 대시보드 반영까지의 처리 속도를 보장함. 데이터가 임상·연구 의사결정에 활용될 만큼 신선한 상태를 의미함. |
| (R&D)생체신호 수집 정확도           | 알고리즘의 입력 데이터 품질을 보장함. 정확도 부족 시 스트레스·불안 추정값이 왜곡되고 위양성/위음성이 증가함.                          |
| (R&D)환자 상태 추출 알고리즘 정확도 | 모델이 임상적 의미가 있는 신호를 안정적으로 구분·예측함을 증명함. 불필요한 경보·누락을 최소화하여 신뢰 확보에 기여함.                 |
| (비R&D) 정책·제도 연계성            | 상용화를 위한 법·제도 기반을 갖추었음을 의미함. 법적 리스크 없이 도입 가능하며, 보험·조달 등 공공 확산 가능성을 판단할 수 있음.       |
| (비R&D)사용자 만족도 및 수용성      | 솔루션의 지속 사용성을 보장함. 사용자 수용성이 높으면 교육·지원 비용 대비 효과가 커지고, 장기 착용·데이터 공유 동의 장벽도 감소함.    |
