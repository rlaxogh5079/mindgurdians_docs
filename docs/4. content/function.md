# 주요 기능

---

## 1. 전문의, 환자가 진료 기록, 처방 기록 조회 기능

전문의와 환자가 처방 기록, 진료 기록을 확인할 수 있습니다.

또한, 환자가 늦은 시간 자해·자살 충동을 느껴 응급 상담 전화를 신청하였을 때, 환자의 사전 동의하에 환자의 진료 기록, 처방 기록을 조회할 수 있습니다.

## 2. 멀티캠 기반 화상통화 기능

일반적인 화상통화 방식은 1개의 캠 만을 이용하여 환자와 진료를 진행하여 환자의 비언어적 표현을 파악할 수 없는 한계점이 있습니다.

해당 한계점을 보완하기 위해 최소 2개의 캠을 동시 송출하는 화상통화 기능이 필요합니다.

환자가 소유한 기기(노트북/스마트폰 카메라, 필요시 추가 캠) 활용을 권장하며, 연구단계에서는 보조 장비를 지원할 수 있습니다.

## 3. 치료 접근 어려움을 최소화하기 위한, 온라인 처방 기능

지방에서 우울증 환자가 약물 치료가 필요한 상황이라면, 온라인 처방을 통해 환자 주변의 약국에서 약을 처방받을 수 있습니다.

## 4. 전문의가 환자의 상태를 실시간으로 확인할 수 있는 모니터링 시스템

전문의가 환자의 상태를 실시간으로 확인하여 위급 상황 시 알림을 받을 수 있습니다.

이전 상태 또한 설정 기간 단위로 확인할 수 있어, 진료를 진행하는데 쉽게 사용할 수 있을 것으로 기대하고 있습니다.

제공되는 데이터는 다음과 같습니다.

| 데이터                      |
| --------------------------- |
| 심박수                      |
| 스트레스 상태               |
| 불안 상태                   |
| 수면 패턴                   |
| 감정 상태(행복, 슬픔, 중립) |

## 5. 의료용 스마트밴드를 지급받기 위해, 기업에 지급을 요청하는 기능

환자가 사용하는 애플리케이션은, 기업에 의료 모니터링 밴드 지급 요청을 할 수 있습니다.

기업은 환자가 정신질환 환자가 맞는지 검증을 진행합니다. 담당 병원에 연락하여 진단 기록 조회를 신청할 수 있습니다.

기업에서 환자의 정신질환 여부가 판단 된다면, 의료용 스마트밴드를 환자에게 지급합니다.

환자는 지급받은 의료용 스마트밴드와 동시에 지급받은 스마트밴드의 고유번호를 애플리케이션을 통해 의료용 스마트밴드를 등록하고 모바일과 연동할 수 있습니다.
